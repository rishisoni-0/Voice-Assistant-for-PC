# Voice-Assistant-for-PC
Use your PC with voice commands, A modern Python voice assistant featuring speech recognition, text-to-speech, and a CustomTkinter GUI. Intelligent desktop assistant built with Python, voice recognition, threaded command execution, and modern UI. Voice-controlled desktop assistant with wake words, GUI controls, and modular architecture.

Before running the code checkout commands.py, config.py to know what are the wake words and commands.

Imports and Installations:
pyttsx3, speech_ recognition, py_audio, customTkinter, setuptools, webbrowser, subprocess, threading, os.

Run the code in Main.py

What happens when you run the code?:
1. Window appears with Name and Buttons (Name can be changeable in gui.py)
2. Click Start Assistant
3. Say the Wake Word to wake up the assistant, you can find and customize wake words in config.py
4. After the assistant replies, Say the command (Commands are in commands.py you can also add any commands if you want)
5. Assistant does whatever command you gave him
6. To Turn Off the assistant say Stop Words (in confing.py) or start/stop the button.
